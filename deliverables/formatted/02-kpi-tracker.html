Setup Instructions & Additional Tabs

KPI Tracker — Google Sheets Setup Instructions

Import the CSV file into Google Sheets, then set up the following tabs and formatting.

Tab 1: Weekly Activity (from the CSV)

This is your main input tracker. Fill it in every week.

Columns:

  • Week: 1-12
  • Date Range: Monday-Sunday dates
  • Warm Messages Sent: Total messages sent to warm contacts this week
  • Follow-Ups Sent: Total follow-up messages sent this week
  • Cold Emails Sent: Total cold emails sent (0 for weeks 1-3)
  • LinkedIn DMs Sent: If using LinkedIn as a channel
  • Discovery Calls Booked: From all sources
  • Discovery Calls Completed: Actually happened
  • Proposals Sent: After discovery calls
  • Free Clients Closed: Case study clients
  • Paid Clients Closed: Paying clients
  • Revenue Closed: Dollar amount closed this week
  • Revenue Collected: Actually received this week
  • Referrals Received: From existing clients
  • Testimonials Collected: Video or written
  • Biggest Win: What went well
  • Biggest Challenge: What was hard
  • Key Learning: What you learned

Suggested formatting:

  • Header row: bold, freeze row 1
  • Conditional formatting on "Warm Messages Sent": green if >= 150 (30/day x 5 days), yellow if 100-149, red if < 100
  • Add a SUM row at the bottom for all numeric columns
